# Delivery Route Optimization – Capstone Assignment

## 📌 Overview

A complete logistics optimization system combining multiple algorithmic paradigms.

## 🧠 Concepts Covered

* Recursion (route cost)
* Greedy (parcel selection)
* Dynamic Programming (knapsack)
* Graph Algorithms (Dijkstra, MST)
* Travelling Salesman Problem (TSP)

## ⚙️ Implementation

Tasks:

1. Input modeling (graph, parcels, constraints)
2. Recursive cost function
3. Greedy + DP delivery planning
4. Shortest path & MST
5. TSP route optimization
6. Performance analysis

## 📊 Results

* Greedy is fast but not always optimal
* DP gives best parcel selection
* TSP becomes infeasible for large inputs

## 📚 Conclusion

Combining multiple algorithms provides practical solutions to real-world logistics problems.
